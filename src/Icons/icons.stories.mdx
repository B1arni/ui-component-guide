## Icon Props
Icons can be used using `i` tag. See the examples.

| Name                   | Type             | default   | Description              |
| ------------           | ---------------  | -------   | -----------------------  |
| className              | text             |           | accepts the class of expected icon. For full list of icons visit [docs](https://icons.getbootstrap.com/#icons)
| style                  | `CSSProperties`  |           | list of css properties as object
